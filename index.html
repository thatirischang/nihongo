古假名（伊呂波歌中出现，现代不再使用）
浊音 — が・ざ・だ・ば 行
半浊音 — ぱ 行
拗音 — きゃ・しゃ・ちゃ … 等
点单字朗读单音

意译

日语由三套文字混写而成

读懂任何一句日语，先理清这三种文字怎么分工。

私わたしは コーヒーを 飲のむ。

「我喝咖啡。」 — 蓝字是汉字（实词），黑字是平假名（语法），红字是片假名（外来词）。

这三套文字不是替换关系，而是分工关系。同一句话里它们各自承担不同任务，少一种都写不成现代日语。

① 平假名 ひらがな — 日语的语法骨架

由汉字草书演变而来。9 世纪左右，日本宫廷女性书写时把汉字一笔写完，逐步简化成今天的圆滑曲线形。所以平假名又叫「女手」（おんなで）。

用来写什么：

  • 所有助词（は・が・を・に・で・と・も・の…）— 这是平假名第一职责。
  • 动词、形容词的活用尾巴（汉字管词根，平假名管词尾变化）。例：食べる（吃，原形）/ 食べた（吃过了）/ 食べない（不吃）— 「食」是汉字不变，后面 る/た/ない 是平假名变形。
  • 没有汉字或汉字罕用的和语词。例：これ（这）、どこ（哪里）、きれい（漂亮）。
  • 给汉字注音（振り仮名 / フリガナ）。例：桜さくら。

学习提示：平假名是入门日语必须最先学完的。哪怕一个汉字都不认识，全用平假名也能写出语法正确的句子，只是看起来像小学一年级课本。

② 片假名 カタカナ — 外来语和强调

由汉字楷书的偏旁截取而来。9 世纪僧人在汉文佛经旁标注训读时，为节省空间只写汉字的一部分（如「阿」取左边的「阝」演变成 ア）。所以片假名笔画硬朗、棱角分明。

用来写什么：

  • 外来词（外来語）— 主要来自英语，也有法/德/葡/中等。例：コーヒー（coffee）、パソコン（personal computer）、アルバイト（来自德语 Arbeit，意为打工）、ラーメン（来自中文「拉麺」）。
  • 拟声词、拟态词。例：ワンワン（汪汪）、キラキラ（闪闪发光）、ドキドキ（心跳怦怦）。
  • 动植物的学名 / 通用名。例：サクラ（樱花）、ネコ（猫，平实写法）。
  • 强调（相当于英文斜体或加粗）。日语没有斜体，要强调一个词常会改写成片假名。
  • 科学术语、品牌名、人名地名（外国）。例：ニューヨーク（New York）、トヨタ（Toyota）。

学习提示：现代日语 10% ~ 20% 的词汇是外来词，全用片假名写。不学片假名，菜单、广告、新闻、IT 文章基本读不懂。

③ 汉字 漢字 — 表意、压缩信息密度

5 世纪从中国传入。明治时代以前日本就一直用汉字。现代日本常用汉字 2136 字（「常用漢字表」，2010 年修订）。

大部分是繁体字（也有日本自造的「国字」如「峠」「畑」「働」），少部分是日式简化字（「国」「広」「学」），跟中国大陆简体字大体兼容但不完全相同。

关键概念 — 一字多读：

  • 音读（音読み）：保留古汉语发音。日本人借汉字时连发音一起借了过来，所以音读听起来「像中文但走了样」。多用于汉语词、复合词。
  • 训读（訓読み）：日本固有词「套」上字义对应的汉字。读音完全是日语本土的，跟中文发音无关。多用于单字词。

山 = やま（训：本土词「山」）/ さん（音：取自中古汉语 sān）
水 = みず（训）/ すい（音）
人 = ひと（训）/ じん・にん（音）
山口 = やまぐち（训训，地名）
富士山 = ふじさん（音音音，山名）

判断规律（粗略）：单字独立出现 → 通常训读；两字以上的复合词 → 通常音读。但例外极多，靠词汇积累。

读音的进阶：浊音 · 半浊音 · 拗音 · 促音 · 长音

学完 50 音清音之后，下面这五个变化系统是日语发音的剩余全部。原理都很简单。

① 浊音 濁音 — 加两点变浊辅音

原理：清辅音 k / s / t / h 声带振动 → 变成浊辅音 g / z / d / b。日语用「浊点」（゛，叫「濁点 / だくてん」，俗称「点点」）标记，加在假名右上角。

か ka → が ga さ sa → ざ za た ta → だ da は ha → ば ba

怎么用：词形上没什么花样，就是词汇本身要不要浊音是固定的，背的时候连同浊点一起背。例：学校（がっこう gakkou）、大学（だいがく daigaku）。

连浊（れんだく）是个稍微高级的现象：两个汉字词组合时，后字的清音会自动变浊。例：手紙 = て + かみ → てがみ（tegami，「信」）；青空 = あお + そら → あおぞら（aozora，「蓝天」）。规律不严格，记词时观察就行。

② 半浊音 半濁音 — 加小圆圈变 p

原理：只对 は 行有效。加一个小圆圈（°，叫「半濁点 / はんだくてん」）→ h 系列变成 p 系列。

は ha → ぱ pa ひ hi → ぴ pi ふ fu → ぷ pu

怎么用：主要出现在外来词（パン pan「面包」、ピアノ piano）、拟声词（ぱちぱち「啪啪」），以及少数和语词的促音后（いっぱい「满满」）。和语词里独立的半浊音很少。

③ 拗音 拗音 — 拼音里的「介音 i」收回去

原理：「i 段假名」（き・し・ち・に・ひ・み・り 及对应浊音/半浊音）后面跟一个小写的 ゃ・ゅ・ょ，把两个音拼成一个音节，不读两拍只读一拍。

き ki + ゃ ya → きゃ kya（不是 ki-ya，是 kya 一拍）
し shi + ゅ yu → しゅ shu
ぎ gi + ょ yo → ぎょ gyo

书写细节：ゃ・ゅ・ょ 必须写得明显小一号，否则就读成两个独立音节。横书时小字偏左下，竖书时偏右上。

怎么用：大量出现在汉字音读里。例：学校（がっこう gak-kō）里的 こう 不算拗音，但 教室（きょうしつ kyō-shitsu）里的 きょ 就是拗音。写真（しゃしん sha-shin，照片）也是。

④ 促音 促音 — 一个停顿的「小っ」

原理：用一个小写的 っ / ッ占一个音节的位置，但不发音，只是停顿一拍，然后把后面那个辅音「憋一下再爆出来」。中文没有这个，类似英语 "hot dog" 中间的短停顿。

学校 = が・っ・こ・う = ga-(停)-kō（gakkō）
切手 = き・っ・て = ki-(停)-te（kitte，邮票）
いっぱい = い-(停)-pai（满满）
ベッド = be-(停)-do（bed，床）

规则：促音只能出现在 k / s / t / p 行之前（罕见 ch）。っ 后面跟什么辅音，那个停顿就「憋住」那个辅音。罗马字写作把后面那个辅音双写（kk / ss / tt / pp）。

听觉特征：说快一点听起来就是「卡顿一下」，但占的时长跟一个正常假名一样（这点很重要 — 别压缩它）。

书写：っ 必须写小，跟拗音的 ゃゅょ 一样，写大了就读成普通的「tsu」。

⑤ 长音 長音 — 元音拉长一拍

原理：元音持续两拍而不是一拍。日语的拍数（モーラ）严格对应假名数 — 长音占两个假名位。长短音区分意义，搞混会误会。例：おばさん（obasan，阿姨）vs おばあさん（obāsan，奶奶）；ゆき（yuki，雪）vs ゆうき（yūki，勇气）。

书写规则（平假名）：不同元音段加不同假名延长：

  • あ段 + あ：おかあさん（妈妈，okāsan）
  • い段 + い：おにいさん（哥哥，onīsan）
  • う段 + う：くうき（空气，kūki）
  • え段 + え（少）或 + い（多）：おねえさん（姐姐）/ せんせい（先生，sensē — 注意是拉长 e 不是读 sen-se-i）
  • お段 + う（多）或 + お（少）：おとうさん（爸爸，otōsan）/ おおきい（大）

注意「えい / おう」在现代口语里普遍读成长音 ē / ō（如 せんせい 念 sensē），但写法上还是 えい / おう。

书写规则（片假名）：统一用一根横线「ー」表示长音，叫「長音符 / 长音棒」。竖书时变成竖线。

コーヒー kōhī（coffee）
ラーメン rāmen
ゲーム gēmu（game）
パーティー pātī（party）

字源速记法 — 会汉字就会假名

每个假名都源自一个汉字。中文学习者认识汉字，看一眼字源就能记住假名形状 — 本站每个假名格子的左上角小字就是它的源汉字。

あ ← 安的草书；ア ← 阿左侧偏旁
の ← 乃的草书；ノ ← 乃第一笔 — 同一汉字的不同切片

规律：平假名多取整字草书（曲线柔），片假名多取部分笔画（棱角硬）。

学习顺序建议

  1. 平假名 50 音（先 1-2 周读熟，写也尽量练）
  2. 片假名 50 音（再 1-2 周。字源大部分跟平假名同源，能省一半力）
  3. 浊音 / 半浊音 / 拗音（学完清音自然就懂规则，不用死背 — 边背单词边遇到边记）
  4. 促音 / 长音（同上，规则简单，难的是听辨长短音区分）
  5. 常用汉字 + 词汇（边读边积累，目标 500 字应付日常 / 2000 字看新闻）
