意译

日语由三套文字混写而成

读懂任何一句日语，先理清这三种文字怎么分工。
本页所有日文都可以点击听发音 🔊

私わたしは コーヒーを 飲のむ。

「我喝咖啡。」 — 蓝字是汉字（实词），黑字是平假名（语法），红字是片假名（外来词）。

这三套文字不是谁替代谁的关系，而是各管一摊。汉字管「意思最重的那个核心」，平假名管「连接和变化的语法部件」，片假名管「从外语借来的词」。少一种都写不成现代日语。

中文里只有一套字，所以这件事一开始会觉得别扭 — 多看几个例子就习惯了。

① 平假名 ひらがな — 日语的语法骨架

9 世纪左右，日本宫廷女性书写时为了快，把汉字一笔写完，慢慢简化成了今天圆滑柔和的样子。所以平假名又叫「女手」（おんなで）。每一个平假名都是某个汉字的草书。

它用来写什么：

  • 所有助词（は・が・を・に・で・と・も・の…）— 中文里没有助词，但日语句子全靠它们粘合，每句话都用得到。
  • 动词、形容词的尾巴。汉字管「核心意思」，平假名管「时态/否定/被动」这些尾巴变化。例：食べる（吃，原形）/ 食べた（吃过了）/ 食べない（不吃）— 「食」是汉字不变，后面 る/た/ない 是平假名在变。
  • 没有汉字、或汉字太冷门不写的和语词。例：これ（这）、どこ（哪里）、きれい（漂亮）。
  • 给汉字注音（振り仮名 / フリガナ）。给小孩、外国人或者生僻字标读音用，写在汉字上方。例：桜さくら。

给初学者：平假名是入门日语必须最先学完的。哪怕一个汉字都不认识，全用平假名也能写出语法正确的句子，就像小学一年级课本那样。学完平假名，相当于拿到了「能拼出任何一个日语词的字母表」。

② 片假名 カタカナ — 外来语和强调

9 世纪的僧人在汉文佛经旁边标读音时，为了省空间只写汉字的一部分笔画（比如「阿」只取左边的「阝」就成了 ア）。所以片假名笔画硬朗、棱角分明，跟平假名的圆润正好相反。

它用来写什么：

  • 外来词 — 主要来自英语，也有法/德/葡/中等。例：コーヒー（coffee）、パソコン（personal computer 缩合）、アルバイト（来自德语 Arbeit，意为「打工」）、ラーメン（来自中文「拉麺」）。
  • 拟声词、拟态词。例：ワンワン（汪汪）、キラキラ（闪闪发光）、ドキドキ（心跳怦怦）。
  • 动植物的通用名（写文章/科普时用）。例：サクラ（樱花）、ネコ（猫）。
  • 强调（相当于英文斜体）。日语没有斜体，想强调一个词就改写成片假名。
  • 外国人名、地名、品牌名。例：ニューヨーク（New York）、トヨタ（Toyota）。

给初学者：现代日语 10% ~ 20% 的词是外来词，全用片假名写。不学片假名，菜单、广告、新闻、IT 文章基本读不懂。学法上和平假名一样，先记字源再背字形。

③ 汉字 漢字 — 表意、压缩信息密度

5 世纪从中国传入。日语用了 1500 多年没断过。现代日本常用汉字 2136 字（「常用漢字表」，2010 年修订），日常报纸、小说、菜单基本都在这个范围里。

大部分是繁体字。也有日本人自造的「国字」（中国没有的字，比如 峠 / 畑 / 働），还有少部分是日式简化字（国 / 広 / 学），跟中国大陆简体字大体兼容但不完全相同。

关键概念 — 一个汉字常常有两套读法：

  • 音读（音読み）：日本人借汉字时连发音也一起借过来了，所以音读听起来「像中文但走了样」。一般用于两字以上的复合词（汉语词）。
  • 训读（訓読み）：把日本本土固有词的发音「套」在意思相同的汉字上，读音完全是日语本土的，跟中文一点关系都没有。一般用于单字独立出现时。

山 = やま（训：本土「山」）/ さん（音：来自中古汉语 sān）
水 = みず（训）/ すい（音）
人 = ひと（训）/ じん・にん（音）
山口 = やまぐち（训训，姓 / 地名）
富士山 = ふじさん（音音音，山名）

大致规律：单字独立 → 通常训读；两字复合 → 通常音读。例外不少，主要靠平时积累。

给初学者：不用一开始就背汉字。学完两套假名再开始接触汉字 — 而且记的时候同时记一个常见词（如学「学」就连「学校 がっこう」一起记），效率最高。

读音的进阶：浊音 · 半浊音 · 拗音 · 促音 · 长音

学完 50 音清音之后，下面这五个变化系统是日语发音的剩余全部。原理都很简单。

① 浊音 濁音 — 加两点变浊辅音

道理很简单：清辅音 k / s / t / h，把声带振动起来 → 就变成浊辅音 g / z / d / b。日语用一个「点点」（゛，正式叫「濁点 / だくてん」）加在假名右上角来标记。

か ka → が ga さ sa → ざ za た ta → だ da は ha → ば ba

怎么用：没有什么活用规则 — 这个词本身要不要带浊点，是死的，背单词时连浊点一起记。例：学校（がっこう gakkō）、大学（だいがく daigaku）。

连浊（れんだく）是个进阶现象：两个汉字词拼在一起时，后字的清音常常自动变浊。例：手紙 = て + かみ → てがみ（tegami，「信」）；青空 = あお + そら → あおぞら（aozora，「蓝天」）。规律不严格，背词时遇到就记。

② 半浊音 半濁音 — 加小圆圈变 p

道理也简单：只对 は 行这一行有效。加一个小圆圈（°，叫「半濁点 / はんだくてん」）→ h 系列就变成 p 系列。

は ha → ぱ pa ひ hi → ぴ pi ふ fu → ぷ pu

怎么用：大量出现在外来词里（パン pan「面包」、ピアノ piano），还有拟声词（ぱちぱち「啪啪」）和少数促音后的和语词（いっぱい「满满」）。和语词里独立的半浊音很少，遇到大概率是借词。

③ 拗音 拗音 — 像中文拼音里的「介音 i」

道理：「i 段假名」（き・し・ち・に・ひ・み・り，以及它们的浊音/半浊音）后面跟一个小一号写的 ゃ・ゅ・ょ，两个字符合拼成一个音节，只读一拍不读两拍。

き ki + ゃ ya → きゃ kya（不是 ki-ya 两拍，是 kya 一拍）
し shi + ゅ yu → しゅ shu
ぎ gi + ょ yo → ぎょ gyo

书写细节：ゃ・ゅ・ょ 必须明显写小一号，否则就读成两个独立音节，意思都变了。横书时小字偏左下，竖书时偏右上。

怎么用：大量出现在汉字的音读里。例：教室（きょうしつ kyō-shitsu）里的 きょ 就是拗音；写真（しゃしん sha-shin，照片）的 しゃ 也是。学完汉字会越来越频繁地遇到。

④ 促音 促音 — 一个停顿的「小っ」

道理：用一个小写的 っ / ッ占一个音节位置，但它不发音，只是停顿一拍，然后把后面那个辅音「憋一下再爆出来」。中文没有这个，最像英语 "hot dog" 中间那短促的停顿。

学校 = が・っ・こ・う = ga-(停)-kō（gakkō）
切手 = き・っ・て = ki-(停)-te（kitte，邮票）
いっぱい = い-(停)-pai（ippai，满满）
ベッド = be-(停)-do（beddo，bed，床）

规则：促音只能出现在 k / s / t / p 行之前。っ 后面跟什么辅音，那个停顿就是在「憋住」那个辅音。罗马字写法是把后面那个辅音双写（kk / ss / tt / pp）。

听觉关键：速度快听起来就是「卡顿一下」，但这一拍跟一个正常假名一样长 — 不能压缩它，否则日本人会听不出来你在说促音。

书写：っ 必须写得明显小一号，跟拗音的 ゃゅょ 一样。写大了就读成普通的「tsu」，意思全变。

⑤ 长音 長音 — 元音拉长一拍

道理：元音持续两拍而不是一拍。日语的「拍」（モーラ）和假名一一对应 — 长音也要占两个假名位置。长短音区分意义，搞混真的会误会。比如：

おばさん（obasan，阿姨）vs おばあさん（obāsan，奶奶）
ゆき（yuki，雪）vs ゆうき（yūki，勇气）

平假名怎么写：不同元音段加不同假名来延长（这是个有点反直觉的规则，多见几次就熟）：

  • あ段 + あ：おかあさん（妈妈，okāsan）
  • い段 + い：おにいさん（哥哥，onīsan）
  • う段 + う：くうき（空气，kūki）
  • え段 + え（少）或 + い（多）：おねえさん（姐姐）/ せんせい（老师，sensē — 写 sensei，读 sensē，最容易踩坑）
  • お段 + う（多）或 + お（少）：おとうさん（爸爸，otōsan）/ おおきい（大）

初学者最容易踩的坑：看到「えい / おう」别按字母读 e-i / o-u — 现代口语里它们普遍念成长音 ē / ō。比如 せんせい 念 sensē 不是 sen-se-i。

片假名怎么写：简单多了 — 统一用一根横线「ー」表示长音，叫「長音符」/ 长音棒。竖书时变成竖线。

コーヒー kōhī（coffee）
ラーメン rāmen
ゲーム gēmu（game）
パーティー pātī（party）

字源速记法 — 会汉字就会假名

每个假名都源自一个汉字。中文学习者认识汉字，看一眼字源就能记住假名形状 — 本站每个假名格子的左上角小字就是它的源汉字。

あ ← 安的草书；ア ← 阿左侧偏旁
の ← 乃的草书；ノ ← 乃第一笔 — 同一汉字的不同切片

规律：平假名多取整字草书（曲线柔），片假名多取部分笔画（棱角硬）。

学习顺序建议

  1. 平假名 50 音（先 1-2 周读熟，写也尽量练）
  2. 片假名 50 音（再 1-2 周。字源大部分跟平假名同源，能省一半力）
  3. 浊音 / 半浊音 / 拗音（学完清音自然就懂规则，不用死背 — 边背单词边遇到边记）
  4. 促音 / 长音（同上，规则简单，难的是听辨长短音区分）
  5. 常用汉字 + 词汇（边读边积累，目标 500 字应付日常 / 2000 字看新闻）
